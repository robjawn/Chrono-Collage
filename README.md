# ChronoCollage

A site where users can learn and share historical photographs. Game features are work in progress. Inspired by [chronophoto](https://www.chronophoto.app/)

## Technologies Used

* HTML 5
* Python
* Django
* Bootstrap
* Heroku
* Amazon Web Services
* bit.io
